## Installation

Larvatus is an extremely simple framework consisting of just one file. You can obtain it using wget:
```console
wget https://raw.githubusercontent.com/baguswijaksono/larvatus/main/larvatus.php
```
## Features
- [x] Routing
- [x] HTTP helpers
- [x] Middleware
